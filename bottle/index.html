Legion Energy Drink
Project Site

Description:
3D soft drink bottle to promote the "Legion" brand.

Path revolves of 3D curves outline of a bottle were used to create surfaces which were then converted into polygons. Both the liquid and glass materials are set to be highly transmissive to give the appropriate appearance.

The label image is exported from Illustrator. The geometry and materials were created in Maya and Blender then exported to glTF and MaterialX data formats respectively,

Model Dimensions:
14cm x 34cm x 6cm (w, h, d).
